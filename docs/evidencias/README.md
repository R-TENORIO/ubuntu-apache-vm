# Evidências do Projeto

Esta pasta contém as capturas de tela (prints) que comprovam a execução de todas as etapas do projeto.

## Instruções para Upload

Para adicionar suas evidências, você pode:

1. **Pela interface web do GitHub:**
   - Clique em "Add file" > "Upload files"
   - Arraste os arquivos de imagem para esta pasta
   - Faça o commit

2. **Pelo comando git (localmente):**
   ```bash
   git add docs/evidencias/*.png
   git add docs/evidencias/*.jpg
   git commit -m "Add project screenshots"
   git push
   ```
