📋 Etapas Realizadas

1 Instalação do VirtualBox

Hipervisor Oracle VirtualBox instalado no host Windows. Interface gráfica configurada no modo Expert para acesso a todas as opções.

2 Download da ISO do Ubuntu

Imagem ubuntu-24.04.3-desktop-amd64.iso baixada do site oficial ubuntu.com e salva localmente para uso como mídia de instalação.

3 Criação e configuração da VM

VM criada com nome Ubuntu_VM, 3710 MB RAM, 2 vCPUs, disco VDI de 25 GB e adaptador de rede em modo NAT.

4 Instalação do Ubuntu 24.04 LTS

Sistema operacional instalado a partir da ISO montada. Usuário rodrigo-tenorio criado. ISO removida após instalação.

5 Atualização do sistema

Executado sudo apt update && sudo apt upgrade -y. Sistema atualizado com todos os pacotes mais recentes.

6 Instalação e configuração do Apache2

Servidor web instalado via sudo apt install apache2 -y. Status confirmado como active (running) via systemctl.

7 Testes de conectividade e acesso

Ping ao Google com 0% de perda. IP 10.0.2.15 identificado. Página padrão do Apache acessada pelo host via HTTP.

8 Personalização do index.html

Arquivo /var/www/html/index.html editado com nano. Esta página é o resultado dessa personalização!
